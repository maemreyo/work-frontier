# Final Items 28–35 Design

The final wave extends the existing Control Room rather than replacing it: Coordinator owns immutable proposal approval, Executive exposes authority-qualified outcomes, and Operator exposes redacted recovery controls. Copilot remains provider-neutral and disabled by default, with a one-way boundary into citation-grounded explanations or ProposedChange drafts.

Security and operations are executable boundaries, not documentation claims. Every remaining catalog harness receives a concrete command and run-scoped artifact. Standard release certification requires 65 blocking receipts for one clean commit, truthful N/A only for undeclared scoped envelopes, an Ed25519 signature, SBOM, exact #539 parity, one writer, and a rollback proof. F1–F4 remain a separate human review gate and cannot be self-approved by implementation code.
